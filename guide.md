# Nvim Tree File Navigation
- **Enter / `o`** → Open file or folder
- **`h`** → Close folder
- **`l`** → Open folder
- **`a`** → Create file
- **`d`** → Delete
- **`r`** → Rename
- **`x`** → Cut
- **`c`** → Copy
- **`p`** → Paste
- **`R`** → Refresh tree
- **`?`** → Show help
